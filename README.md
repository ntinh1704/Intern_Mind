# Intern_Mind – Todo List

Mini project: Todo List using React + TypeScript + SCSS + Redux Toolkit.

## Tech Stack

- Vite + React + TypeScript
- SCSS (Sass)
- Redux Toolkit + React Redux

## Features

- Thêm công việc (Todo)
- Sửa công việc
- Xóa công việc
- Tìm kiếm công việc theo từ khóa
- Đánh dấu hoàn thành / chưa hoàn thành

## Run Project

```bash
npm install
npm run dev
